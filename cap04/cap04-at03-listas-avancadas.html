Seção 3 - Glossário

HTML
Linguagem usada para criar páginas.
Significa HyperText Markup Language.
CSS
Usado para estilizar páginas.
JavaScript
Linguagem usada para interatividade.
DOM
Representa os elementos da página.
Pode ser alterado com JavaScript.
API
Interface de Programação
Faz comunicação entre sistemas.
HTTP
Protocolo HTTP
Usado para acessar páginas na internet.
URL
Endereço de um site.
DNS
Traduz nomes de sites para IP.
